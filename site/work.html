mkdoc

  • Commonmark
  • Nodejs
{{#markdown}} Fast, versatile and modular toolkit for processing markdown documents as streams featuring a suite of command line interfaces and a programmatic API that includes bindings to many modules for common tasks. Use the toolkit to generate dynamic readme documents, simple static web pages or generate API documentation. The [mkcli](https://github.com/mkdoc/mkcli) module is a particularly useful tool for defining command line interfaces as markdown and generating language-agnostic program definitions, help documents, man pages and zsh completion scripts. {{/markdown}}
